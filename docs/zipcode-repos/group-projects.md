## Why Excluded

| Criterion | Group projects | Individual labs |

| Collaboration | Multi-person git history | Solo JJ commits |
| Scope | Multi-week capstones | Single-session exercises |
| Lifecycle | Ongoing maintenance (Notification-Rake) | Frozen at cohort end |
| Branch strategy | Standard git flow | as-is / stock / refresher triplets |
